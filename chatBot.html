AI Assistant for Studying

Чат-бот для ВКонтакте на основе GigaChat, созданный в рамках проектной практики. Бот помогает студентам разбираться в сложных темах, объясняет материал простым языком и отвечает на учебные вопросы.

Возможности бота

  • Объясняет сложные темы простым языком с примерами из жизни
  • Помогает разобраться с учебными задачами
  • Отвечает на вопросы по любым учебным предметам
  • Обращается к пользователю по имени
  • Поддерживает команды /start и /help

Стек технологий

  • Python 3.14
  • vkbottle — библиотека для работы с VK API
  • GigaChat API — языковая модель от Сбера
  • python-dotenv — безопасное хранение токенов

Архитектура

Бот работает через Long Poll API ВКонтакте: получает сообщения пользователей, формирует промпт с системным контекстом и отправляет его в GigaChat. Полученный ответ очищается от Markdown-разметки и возвращается пользователю.
