Eltech

Мобильное приложение для Московского Политеха

Eltech — кроссплатформенное мобильное приложение для студентов и преподавателей Московского Политеха. Приложение объединяет в себе расписание занятий, личный кабинет и систему управления обучением Moodle в едином удобном интерфейсе для iOS и Android.

Ключевая информация:

  • Руководитель проекта: Клепиков Станислав
  • Платформа: Android и IOS
  • Технология: Kotlin Multiplatform (KMP)
  • Статус: В активной разработке, первичная версия опубликована в RuStore
  • Исходный код: Открытый (open-source), репозиторий на GitHub
